
# Calculadora IMC

Este projeto é uma calculadora de Índice de Massa Corporal (IMC) desenvolvida em HTML, CSS e JavaScript. Ela permite que o usuário insira seu peso e altura para calcular o IMC e exibe uma mensagem indicando o estado de saúde conforme os parâmetros da OMS.

## Funcionalidades

- Cálculo automático do IMC ao informar peso e altura.
- Exibição do valor do IMC com duas casas decimais.
- Mensagem personalizada sobre o estado de saúde: abaixo do peso, peso ideal, sobrepeso, obesidade moderada, severa ou mórbida.
- Interface moderna e responsiva.

## Como usar

1. Abra o arquivo `index.htm` em seu navegador.
2. Informe seu peso (em kg) e altura (em metros).
3. Clique em calcular para ver o resultado do IMC e a mensagem correspondente.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)

## Estrutura do projeto

- `index.htm`: Página principal da calculadora.
- `css/styles.css`: Estilos da interface.
- `js/scripts.js`: Lógica de cálculo e interação.
- `img/`: Imagens utilizadas na interface.

## Sobre o IMC

O IMC é um índice utilizado para avaliar se o peso está adequado em relação à altura. Os parâmetros são:

- Abaixo de 18.5: Abaixo do peso
- 18.5 a 25: Peso ideal
- 25 a 30: Sobrepeso
- 30 a 35: Obesidade moderada
- 35 a 40: Obesidade severa
- Acima de 40: Obesidade mórbida
